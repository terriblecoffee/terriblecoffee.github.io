Typography in HTML and CSS

General Rules

  • Heading tags range from h1 through to h6 and you shouldn't skip levels of headings.
  • Ensure your font selections can be rendered on every browser, or provide secondary and tertiary options.
  • Don't crowd your page. Ensure there is sufficient whitespace inbetween sections, headings, and even between letters and words.

Attributes

Attribute Name Description
font-family Font Family Changes the font of text, can provide backups in the arguement.
font-size Font Size Sets the size of the text.
font-weight Font Weight Changes the thickness of text, from thin to bold.
font-style Font Style Add italics or underlines to text.
letter-spacing Letter Spacing Changes the space between each letter, also known as kerning.
word-spacing Word Spacing Changes the space between words in a string.
color Font color Changes the color of the text
text-align Text Align Aligns the text to the left, center, or right within it's parent.
text-transform Text Transform Sets the text to uppercase, lowercase, or capitalized.
